## Key Properties

### 1Land
A knockdown effect state identifier for tracking creature stumbling.

### 1LandLow
A minor knockdown or stumble state for creatures that lose balance briefly.

### 1None
A neutral state indicator meaning no active knockdown or stun effect is occurring.

### 1Stumble
A state where a creature loses footing momentarily but doesn't fully fall down.

### AIFeralSense
How much sharper an enemy's smell or detection senses become.

### AIGroupCircle
Whether creatures coordinate to surround prey in formation.

### AINoiseSeekDist
How far away creatures search for sounds you make.

### AIPathCostScale
How much zombies prefer or avoid walking through this area.

### AISeeOffset
How much creatures boost or reduce their vision abilities.

### AITarget
Who or what a creature prioritizes attacking.

### AITarget-1
The first behavior rule that controls what this creature attacks.

### AITarget-2
The second targeting rule that controls what this creature attacks.

### AITarget-3
The third targeting rule that controls what this creature attacks.

### AITarget-4
The fourth targeting rule that controls what this creature attacks.

### AITarget-5
The fifth available target that the AI can aim at or pursue.

### AITask
All the actions creatures can do in sequence to hunt and attack.

### AITask-1
Primary AI behavior task assigned to the entity.

### AITask-2
The second behavior pattern this creature follows during gameplay.

### AITask-3
The third behavior pattern this creature follows during gameplay.

### AITask-4
The fourth behavior pattern this creature follows during gameplay.

### AITask-5
The fifth action a zombie or creature should do in sequence.

### AITask-6
The sixth action a zombie or creature performs in its behavior sequence.

### AITask-7
The seventh behavior action for creatures during combat or exploration.

### AITask-8
The eighth behavior action in a creature's action sequence.

### AITask-9
Special behavior pattern a creature can execute.

### ActionExpBonusMultiplier
How much extra experience you earn using this particular item compared to normal.

### ActivateEvent
What happens when a block is activated by the player.

### ActivateObject
The part of the item that turns on when you activate it.

### ActivateSound
The sound that plays when you use this block.

### ActivateTime
Delay before a block activates after being triggered.

### ActivatedAnimBool
The animation setting that plays when a block becomes active.

### ActivatedAnimState
The visual state a block changes to when activated.

### ActivatedAnimTrigger
The action signal that starts the activation animation.

### ActivationDistance
How far away the block can be activated from.

### ActivationEmissive
Which part of an item's picture glows brighter when you activate or use it.

### ActivationEvent
The special event that triggers when this particular block is activated or used.

### ActivationTransformToHide
Which visual parts disappear when an item is activated.

### ActiveRadiusEffects
Bonuses that apply to players standing near this block.

### AllowAllRotations
Lets you rotate this block in all possible directions including upside down and sideways.

### AllowUnEquip
Whether you can remove this item from your hands once you've equipped it.

### AllowedRotations
Which rotations are allowed when placing the block.

### Allowed_upgrade_items
Which materials can be used to improve or enhance this item.

### AltItemTypeIcon
An alternate picture symbol shown in the menu for this item's secondary function.

### AltMats
Which alternate appearance textures the zombie can randomly use.

### AlwaysActive
Whether a block runs constantly without needing activation.

### AmmoItem
The type of ammunition this weapon or block uses to fire.

### AnimType
What kind of motion or movement this item uses.

### AnimWait
How long the animation plays before the action takes effect when using an item.

### Archetype
What basic type of entity this is, like which trader or character template it follows.

### ArmorGroup
The armor set name that gives bonuses when wearing multiple pieces from the same collection.
